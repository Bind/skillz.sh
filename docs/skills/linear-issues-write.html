linear-issues-write

Create and update Linear issues via CLI (write operations)

bunx skz add linear-issues-write
Required env: LINEAR_API_KEY
Entry points: create-issue, update-issue

Setup Required

Get your API key from Linear Settings > API > Personal API keys

Dependencies

  • @linear/sdk ^29.0.0

Overview

CLI tools for creating and updating Linear issues. Requires LINEAR_API_KEY set in /.env or exported in the environment.

Prerequisites

  • bun runtime installed
  • LINEAR_API_KEY set in /.env or environment
